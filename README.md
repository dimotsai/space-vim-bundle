This project would build all essential packages for space-vim and pack them
as a standalone tarball that could be installed offline.

Included Packages
-----------------

1. neovim 0.3.8
2. ccls 0.20190314.1
3. bear 2.4.0
4. python 3.7.3
5. space-vim master
6. all plugins in space-vim layers

Build Requirements
------------------

### Arch Linux
1. gcc
2. make
3. cmake
4. patchelf
5. lib32-gcc-libs
6. ncurses
7. ncurse5-compat-libs (AUR)
