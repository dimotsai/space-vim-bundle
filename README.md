This project builds all essential packages for space-vim and pack them
together as one single standalone executable that can be installed without internet.

Included Packages
-----------------

1. neovim 0.7.2
2. ccls 0.20210330
3. bear 2.4.3
4. python 3.9.13
5. space-vim master
6. all plugins in space-vim layers

Build Requirements
------------------

* Docker: make docker
